## Manual Installation Steps

1. **Install Tampermonkey Extension**
   - Chrome/Edge: https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo
   - Firefox: https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/

2. **Open Script URL**
   - Click one of the installation links above
   - OR copy the URL and paste into Tampermonkey "Install from URL"

3. **Install Script**
   - Tampermonkey will open
   - Click "Install" button

4. **Verify Installation**
   - Visit BigPicture page
   - Check console for: `[BigPicture Russian Translation] Script loaded`
   - Check UI displays in Russian

## Which Version Should I Use?

- **Simple Version**: Start here - works for most cases
- **Advanced Version**: Use if you see old English text after updates (caching issues)
- **External Version**: Use if you want to load translations from external source (GitHub/CDN)
